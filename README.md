## 🧪 Real Backend Challenges I Solved

This project involved **serious backend debugging**, including:

- Fixing MongoDB `E11000 duplicate key error` caused by compound indexes
- Handling `null` values breaking unique constraints
- Designing **persistent profile forms** (similar to social media edit profiles)
- Debugging repeated **500 Internal Server Errors**
- Fixing password reset flows after breaking existing auth logic
- Managing JWT expiry, refresh behavior, and session persistence
- Handling async race conditions in form submissions
- Fixing production-only bugs that did not appear locally
- Resolving deployment configuration mismatches
- Refactoring schema designs after real data conflicts
- Preventing accidental data overwrites in update APIs

These problems **cannot be avoided** in real backend roles —  
and I faced them directly.
